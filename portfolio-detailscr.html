Overview of CrossOver for macOS

Run Microsoft Windows software on your Mac without buying a license, rebooting, or using a virtual machine. It makes it easy to launch Windows apps natively from the dock and integrates macOS functionality like cross-platform copy & paste and shared file systems to your Windows programs.

Easy Installs! Many Windows titles can be installed using our CrossTie technology. If you have CrossOver installed, and have the installer for your Windows software, click the button in our Compatibility Database, and it will automatically install your application for you!
